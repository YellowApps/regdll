RegDLL

RegDLL - удобный инструмент для регистрации DLL в системе.

ИНСТРУКЦИЯ:
	1) Поместите нужные DLL в папку с RegDLL.
	
	2) Запустите от имени администратора:
		regdll64.exe -> для 64-битных DLL;
		regdll32.exe -> для 32-битных DLL.
	
	3) Дождитесь регистрации всех DLL и перезагрузите ПК.
